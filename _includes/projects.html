Current Research, Areas of Interest and Projects

GtaAssignment Application

  • A system based around the idea of automating the process of selecting graduate teaching assistants (GTA) for classes.

This project uses an Integer Linear Program (ILP) to optimally match GTAs to classes. By adding GTA and Instructor preferences as factors in the objective function and using required skills and GTA experience as constraints, the system finds an optimal solution.
